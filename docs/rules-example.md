# フォーマットルール サンプル

このファイルを `--format` オプションで指定すると、AIへの指示として使われます。
必要なルールだけ残して、不要な行は削除して使ってください。

---

## 分野ツリーのルール

- 分野名は日本語で統一する
- 分野の階層は最大3階層まで
- 汎用的すぎる分野名（「その他」「雑記」など）は作らない

## ページ紐づけのルール

- 1ページあたりの分野リンクは最大3つまで
- タイトルが `.icon` で終わるページ（例: `hr.icon`）は候補に含めない
- 設定ページ（`settings` など）は候補に含めない

## 親リンクのルール

- 3つ以上の分野がまとめられる場合のみ親リンクを提案する
- 親リンク名は具体的かつ簡潔にする
